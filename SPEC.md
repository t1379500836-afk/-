# AutoWorker - 自动化操作工具

## 项目概述
Windows桌面自动化工具，支持录制和回放鼠标、键盘操作，具备坐标拾取、工作流编辑、全局热键控制等功能。

## 技术栈
- 语言：Python 3.x
- GUI框架：PyQt6
- 自动化库：pyautogui + pywin32
- 热键库：keyboard

## 功能模块

### 1. 操作节点管理
| 类型 | 参数 |
|------|------|
| 鼠标操作 | 单击类型(左/右/中)、X坐标、Y坐标 |
| 键盘操作 | 按键码、是否组合键(Ctrl/Alt/Shift) |
| 时间间隔 | 延迟毫秒数 |

### 2. 坐标拾取
- 点击"拾取坐标"进入拾取模式
- 鼠标变为十字光标
- 按F4键捕获当前坐标
- 自动填入参数框

### 3. 工作流编辑
- 列表展示所有步骤
- 上移/下移调整顺序
- 编辑/删除操作
- 设置循环次数或无限循环

### 4. 执行控制
- 全局热键：F6启动、F7停止、F8暂停/继续
- 后台线程执行，不阻塞UI
- 执行进度实时显示
