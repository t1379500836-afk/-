# 轻松两点 - 自动化操作工具

## 项目概述

Windows桌面自动化工具，支持录制和回放鼠标、键盘操作，具备坐标拾取、工作流编辑、全局热键控制等功能。

## 技术选型

| 类别 | 技术 | 说明 |
|------|------|------|
| 语言 | Python 3.x | 开发语言 |
| GUI框架 | PyQt6 | 跨平台GUI框架 |
| 自动化库 | pyautogui + pywinini | 鼠标/键盘操作 |
| 热键库 | keyboard | 全局热键监听 |
| Excel处理 | openpyxl | 读取Excel数据 |
| 剪贴板 | pyperclip | 文本输入辅助 |
| 打包工具 | PyInstaller | exe打包 |

## 功能描述

### 1. 操作类型
| 类型 | 参数 | 说明 |
|------|------|------|
| 鼠标左键单击 | X坐标、Y坐标 | 移动并左键点击 |
| 鼠标右键单击 | X坐标、Y坐标 | 移动并右键点击 |
| 鼠标中键单击 | X坐标、Y坐标 | 移动并中键点击 |
| 键盘操作 | 按键码、Ctrl/Alt/Shift | 按键或组合键 |
| 文本输入 | 文本内容或Excel文件 | 支持空格分隔多行文本或Excel列数据 |
| 延时 | 毫秒数 | 步骤间延迟 |

### 2. 坐标拾取
- 点击"拾取坐标"按钮进入拾取模式
- 鼠标变为可捕获状态
- 按F4键捕获当前坐标
- 自动填入X/Y输入框

### 3. 工作流管理
- 多个工作流切换/创建/删除
- 步骤列表：上移/下移/编辑/删除
- 循环设置：固定次数或无限循环

### 4. 执行控制
- 全局热键：F6启动、F7停止、F8暂停/继续
- 后台线程执行，不阻塞UI
- 实时进度显示

## 操作方法

### 1. 添加操作步骤
1. 选择操作类型（鼠标/键盘/文本/延时）
2. 设置参数（坐标/按键/文本/延时）
3. 点击"添加操作"
4. 自动在非延时操作后添加1000ms延时

### 2. 使用坐标拾取
1. 点击"拾取坐标"按钮
2. 移动鼠标到目标位置
3. 按F4键捕获坐标

### 3. 文本输入配置
- **直接输入**：文本框中输入多个内容，用空格分隔，每次循环输入一个
- **Excel输入**：选择Excel文件→设置列号→设置起始行→循环读取

### 4. 执行工作流
1. 点击"开始"按钮或按F6
2. 使用F8暂停/继续
3. 使用F7立即停止
